# v0.1.0

发布日期：2026-03-08

## 下载

- Android APK: `OneWord-Android-v0.1.0.apk`
- macOS DMG: `OneWord-macOS-v0.1.0.dmg`
- 安装说明: [docs/INSTALL.md](../INSTALL.md)

## Highlights

- 初始化 Android / iOS / macOS 三端工程
- 接入今日诗词接口与本地 SQLDelight 缓存
- 实现海报式首页、全文展开、刷新与离线回显
- 支持动态主题取色与主题持久化
- 支持 macOS DMG 打包
- 提供 iOS Simulator 一键重置与运行脚本

## Included

- Kotlin Multiplatform + Compose Multiplatform 共享 UI 与业务层
- Android 12+ 应用入口
- iOS 13+ 容器工程与数据库路径修复
- macOS Desktop 应用入口、菜单栏、图标与打包配置
- GitHub Actions CI

## Notes

- iOS 首屏已补齐安全区处理
- iOS `Info.plist` 已补齐 `CADisableMinimumFrameDurationOnPhone`
- iOS SQLDelight Native Driver 已改为 `name + basePath` 形式配置数据库
